How to use

You can use this site Mai Sung to transfer your data(files) to your friend online without download or install any applications. In this page, you can learn how to use this site to create room and join room.

Steps for Sender

  • Click icon on the navbar or click here to create room.
  • Inside of create room page, press Create room to generate room Id.
  • Send the generated Room Id to your friend and wait for he(she) input this code from his(her) side.
  • If both of site were connected, it will ring notification sound and then you can start sending files.

Steps for Receiver

  • Click icon on the navbar or click here to join room that sender created.
  • Enter the Room Id from the sender in the input box.
  • If both of site were connected, it will ring notification sound and then you will receive files from the sender shared to you.

Warning : You can't send mp4, json and larger files for yet, we'll fix as soon as possible.
